# Folders: Front-End and Back-End
 - __Back-End__: 
  * Dev: 
      * Change MongoDB URL.
      * Execute: npm install
      * Run: npm run dev
  * Production: 
      * Change MongoDB URL.
      * Execute: npm install
      * Run: npm run production
      
 - __Front-End__:
  * Execute: npm install
  * Run: npm start
